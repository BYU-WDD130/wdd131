Red Rose Desserts – Website Plan

Subject/Entity of Website

Red Rose Desserts – A local artisan bakery specializing in elegant, handmade desserts such as 3D gelatin cakes, cupcakes, mousse, and seasonal treats.

Purpose of the Website

  • Promote Red Rose Desserts to a wider audience.
  • Display a menu of available dessert products.
  • Allow customers to place custom orders through a contact form.
  • Share testimonials and images from happy clients.
  • Provide store hours, location, and contact details.

Target Audience

  • Local customers looking for custom desserts.
  • Event planners for weddings, birthdays, holidays.
  • Dessert lovers who appreciate artistic treats.
  • Users browsing on both desktop and mobile devices.

Scenarios

  • “I need to find a unique cake for my daughter’s quinceañera.”
  • “I’m planning a baby shower and want to see dessert options and pricing.”

Main Goals

  • Create a mobile-friendly, accessible website using responsive design.
  • Show a visually appealing gallery of desserts using optimized images.
  • Use JavaScript features (e.g., menu filter, form validation, localStorage).
  • Allow users to place inquiries via a form.

Page Structure

Page Purpose
Home Welcome message, featured product image, intro to bakery.
Menu Display a categorized list of desserts with filtering.
Order/Contact Custom order form and contact information.

Design Considerations

  • Mobile-first layout using flexbox and grid.
  • Simple navigation bar on all pages.
  • Color scheme:
    • Primary: #c2185b (Rose Pink)
    • Accent: #ffffff (White)
    • Background: #fff8f5 (Light Cream)
    • Accent1: #5d4037 (Deep Brown)
  • Typography:
    • Headers: Handwritten-style (future use: 'Dancing Script' or similar)
    • Body: 'Open Sans', sans-serif
